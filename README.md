# 项目介绍
这是一个烂到家的项目，只是为了满足自己的需求，没有任何的优化和完善。

# 功能
可以以优雅的方式插入文件内容到聊天记录中
在插入图片时可以自动切换为支持视觉的模型
可以在托盘上运行
可以自动生成历史记录（太多太长的话json大小会爆炸，自己删）

## 添加窗口
可以ctrl+v添加剪切板的内容，支持图片和文本
可以拖入文件来添加文件内容，仅支持文本文件

## 历史记录窗口
点击切换到对应对话
选中后按del删除对应对话

## 设置窗口
三个框为：
纯文本模型（支持视觉也没事）
视觉模型（不支持视觉会报错）
标题生成模型（要支持前缀补齐）

# 注意事项
复制model-example.json到models.json
修改apikey为自己的apikey
