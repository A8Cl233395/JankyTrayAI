项目介绍

本项目为满足个人使用需求而开发，未进行性能优化或功能完善，仅保证基本可用。

功能说明

- 支持以结构化方式将文件内容插入聊天记录。
- 插入图片时，自动切换至支持视觉输入的模型。
- 支持程序最小化至系统托盘运行。
- 自动保存对话历史记录（格式为 JSON）。
  - 注意：对话内容过长或数量过多可能导致文件体积过大，请定期手动清理。

添加窗口

- 支持使用 Ctrl+V 粘贴剪贴板内容，可识别文本与图片。
- 支持拖拽文件添加内容，仅接受纯文本格式文件。

历史记录窗口

- 单击条目可切换至对应对话。
- 选中条目后按 Delete 键可删除该对话记录。

设置窗口

需配置以下三项模型参数：

1. 纯文本模型 —— 支持处理普通文本，是否具备视觉能力不影响使用。
2. 视觉模型 —— 必须支持图像输入，否则操作将失败。
3. 标题生成模型 —— 需支持前缀补全（prefix-completion）功能。

使用前准备

1. 将 model-example.json 复制并重命名为 models.json。
2. 在 models.json 中替换 apikey 字段为您的有效密钥。
3. 若使用视觉识别功能，需提供 Azure Computer Vision 的 api_key 与 endpoint。
   - 如无相关资源，请保持“自动视觉模式”开启，以避免运行错误。
